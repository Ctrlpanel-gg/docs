# Introduction

Welcome to the CtrlPanel.gg Beta version documentation. Here you'll find guides for installation, updating, and other useful information about public beta testing.

## What is Beta?

Public beta testing helps us identify possible issues or even critical bugs before the full stable release. This ensures that obvious errors don't make it into the stable release and cause problems for users.

## When does beta testing take place?

We announce the opening of public beta testing on our Discord server - this usually happens after we've finished development of a new version and are ready for release.

## Who can participate and how?

We welcome everyone who wants to help us test the upcoming version before its full release. No special skills are required - basic knowledge of CtrlPanel, minimal hands-on experience, and the desire to help the project grow is all you need.

In the beta testing announcement we always provide information on where to find the necessary materials, where to report bugs, where to get help, and how to update or install the beta version. We recommend keeping an eye on our Discord server for updates.
